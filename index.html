DrumBeat 陣令

Concept

　　古代作戰時以「擊鼓」來發號施令，讓士兵們聽從鼓聲進行陣形的變化；我們讓使用者透過擊鼓的行為以及數位視覺的呈現，從而體驗這樣的過程。


Interactive

EarthBeat 星跳

Concept

　　每一個大自然的生物都是地球的一個心跳，而人類不斷的開發環境使得地球的心跳失去了色彩，於是我們希望透過關燈這樣簡單又貼近生活的動作來傳達環境保育的重要性。


Interactive

　　以心電圖的方式呈現整個地球的心跳。當使用者將電源開關開得越多，地球的心跳會出現越來越多工廠、電線桿等人為建築，並且失去了顏色；反之，當電源開關關得越多，地球的心跳又會回到綠意盎然的樣貌。

Fire x Escape

防災教育遊戲之研究與創作－以火災逃生為例

防災教育｜數位遊戲學習｜教育遊戲設計｜火災應變表現


Concept

　　有鑑於教育遊戲多數有乏味以及黏著度低的問題，我們期望打造一個有商業遊戲模式的教育遊戲，藉以探討提升遊戲黏著度的可能。遊戲題材則圍繞著火災逃生，除了融入火災逃生的教材以外遊戲本身也依此設計，也希望使用者在沉浸遊戲的過程中能獲得隱含其中的知識。


Interactive

　　遊戲玩法為第一人稱的3D探索遊戲，以「模擬真實」的角度實現火場於遊戲關卡中，並輔以體力、道具、商店及排行榜等系統，提高使用者的遊戲體驗。使用者需面對火場狀況進行逃生選擇，以逃出火場為目的來過關。

認識消化道

Introduction

　　科教館常備展「認識消化道」中的其中一項裝置，透過人形立板上的消化道以及64顆LED跑燈效果，演出食物進到人體後的旅程。


Interactive

　　玩家可透過小螢幕解說了解食物進入人體後如何移動（依序是食道、胃、小腸、大腸），而一旁的人形立板也會在對應的部位作LED跑燈效果演出，並不斷重覆演出此過程。

漢字遊

Introduction

　　一款以學習漢字為主要目的遊戲APP。遊戲學習內容為與動物相關的漢字，玩家可以透過遊戲過關和收集的要素來掌握對漢字的認識。


Interactive

　　進入關卡後系統會隨機釋出一種動物的漢字，玩家必須找到並且透過觸碰的方式來進行互動，也就是手指點擊畫面中正確對應於系統所出的動物漢字，且要在遊戲時間內找到系統出的所有漢字。遊戲提供圖鑑讓玩家閱覽找到的漢字說明及動物繪圖，也讓玩家體驗到收集的成就感。

叢戰巴龍

Introduction

　　玩家在遊戲中將扮演易大師與小龍及巴龍對抗，利用Kinect體感裝置偵測玩家「砍劈」的動作，來增加遊戲的互動性，並帶給玩家有自己在與敵人對戰的感受。


Interactive

　　遊戲一開始會處在待機畫面，當Kinect偵測到玩家時便會開始遊戲。遊戲一開始提供一個教學關卡來幫助玩家練習揮砍的動作以及時機點，接著便進入與巴龍的實戰，隨著玩家的Combo數越高，揮砍的節奏也會出現的越快。在玩家必殺技量表充滿時，只要做出畫面指定動作就可以施展易大師的必殺技。
